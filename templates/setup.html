Setting up your endpoint

Installing this endpoint

Please follow the installation instructions in the project GitHub repository to find out how to install the endpoint.

Adding head tags to your site

To set up this endpoint, you will need to add two HTML tags to the head of every page on your site:

<link rel="hub" href="your_hub_url"/> <link rel="self" href="web_page_url"/>

Please substitute the above hub href value with the actual values or your hub. The self value should be equal to the HTML page being viewed.

You will need to send a POST request to your hub every time you create new content on your site. This is necessary so that the hub can immediately notify any subscribers to a page that new content has been added.

Subscribing to content

Please refer to the "How to Subscribe" section of the IndieWeb wiki WebSub tutorial for informtaion on how to subscribe to a WebSub hub.
